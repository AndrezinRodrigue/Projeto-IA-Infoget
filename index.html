SOBRE O PROJETO

O projeto surgiu como uma alternativa para a feira de ciências da Escola Estadual Comendador Zico Tobias. Uma iniciativa vinda dos alunos do projeto Infoget com a orientação do professor André Rodrigues.
O projeto visa mostrar os conceitos básicos da inteligência artificial de forma simples onde as pessoas poderão ver desde as vantagens até os riscos que este tipo de tecnologia pode trazer, além disso, poderão interagir com uma das formas mais simples dessa tecnologia.
